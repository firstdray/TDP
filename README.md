# Digital Portfolio Backend

Бэкенд-часть цифрового портфолио для ГАПОУ ТСПК. Разработана на базе фреймворка Nest.js с использованием TypeScript и PostgreSQL.

## Описание проекта

Цифровое портфолио — это платформа, позволяющая студентам ГАПОУ ТСПК демонстрировать свои навыки, учебные проекты и профессиональный потенциал будущим работодателям. Бэкенд обеспечивает хранение данных, аутентификацию, авторизацию и бизнес-логику приложения.

### Основные возможности

- Регистрация и аутентификация пользователей (JWT)
- Разграничение ролей: студент, преподаватель, администратор
- Управление студентами и преподавателями
- Работа с отделениями и группами
- Ведение портфолио студентов
- Безопасное хэширование паролей (bcrypt)
- Универсальный обработчик EntityPipe для поиска сущностей

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Фреймворк | Nest.js |
| Язык | TypeScript |
| База данных | PostgreSQL |
| ORM | TypeORM |
| Аутентификация | JWT (JSON Web Token) |
| Хэширование паролей | bcrypt |
| Среда выполнения | Node.js |
